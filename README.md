## What This Is

NAST is a diagnostic framework for analyzing how market logic (A2 utility reduction) systematically fails to account for the costs of destroying time-bound, irreproducible entities — ecosystems, languages, community ecologies, cultural practices.

The core claim: management obligations arise from **benefit relationships**, not property rights. Whoever benefits from a temporal being's continued existence holds a stewardship office — non-transferable, non-inheritable, non-waivable. This is not a moral argument. It is a structural derivation from the benefit relationship itself.

NAST does not prescribe a target state. It identifies a boundary: the irreversibility threshold (D2) below which recovery becomes impossible. Stewardship means keeping the system above that threshold and passing it to the next generation.

## What NAST Diagnoses

**A2 blind spots**: When an entity satisfies any of:
- α (non-discountable): meaning distributed across infinite time horizon
- β (non-operable): maintenance cannot be monetized without destroying it
- γ (non-identifiable): A2 cannot build an input-output mapping

A2 cannot state the entity — not "too valuable to destroy," but **absent from A2's grammar**.

**Cognitive contamination (ND11)**: A2 uses proxy metrics to simulate having captured uncapturable costs. The proxy's existence closes the inquiry. Carbon markets, ESG scores, vacancy taxes — the problem is not inaccurate pricing, it is that pricing itself forecloses asking whether pricing is appropriate.

**A2→A3 positive feedback loop**: A2 prices the unquantifiable → A3 trades the proxy → market prices validate A2's framing → the underlying loss continues in silence.

---

## What NAST Does Not Do

NAST does not provide a target state ("nature should look like X"). It identifies a boundary line, not an ideal.

NAST does not solve the scalability problem. Embedded bearing — the cost-bearing structure required for stewardship — requires small-scale, non-anonymous, non-delegable communities. Anonymous mass society is A2's carrier, not stewardship's. This is a confirmed boundary (SORRY-NAST-2), not a design flaw.

The diagnostic-prescriptive discrete boundary theorem (§10.2): any framework that precisely diagnoses A2 must use language A2 cannot translate. Any prescription that can scale must enter A2-translatable language and risks absorption. There is no middle ground.
