## 一、判定层：再生产能力是否涉及时间性存在

### 1.1 分析对象

**不是**：个别家庭的生育决策（个人选择层，不在NAST范围）

**是**：社会的人口再生产能力——作为支撑社群持续存在的时间性结构。

满足N6不可再现性：
- 一旦人口结构跨越ℋ_b（迟滞区），恢复需要跨代周期，无法通过政策逆转
- 重建后的社群（通过移民等替代路径）与原社群在文化/知识/操作知识层面信息结构不等价
- CCST实证：韩国TFR 0.72，日本TFR 1.15——均已进入σ_b激活区间，TIFM有数据

分析对象：**社会再生产能力的可持续性**，是NAST意义上的时间性存在。

---

## 二、诊断层：NAST命题映射

### 2.1 N3：再生产成本是原债，不是个人成本

N3的原债内容：受益者对时间性存在负有维护义务，不可货币化清偿，不可通过代理履行而消灭。

**映射**：任何从社会再生产中受益的人（使用劳动力、享受社群网络、依赖代际知识传递的所有人），对社会再生产能力负有原债。

**关键推论**：再生产成本（托育、教育、医疗、养老）是原债履行的必要手段，不是个体自愿选择的私人成本。

### 2.2 N5：再生产成本市场化 = 国家化悖论的私有化变体

N5指出国家化是成本外部化的制度伪装。市场化是另一种形式：

```
国家化：国家以"全民代表"名义汲取资源，
        将维护成本转嫁给具体社区
市场化：市场以"效率最优"名义定价资源，
        将再生产成本转嫁给个体承担者
```

两者结构同构，只是伪装语言不同——一个用公共话语，一个用A2话语。

**市场化做了什么**：
- 把托育/教育/医疗/养老从集体原债义务变成市场商品
- 使"有剩余的人"（资本持有者）可以购买再生产（A6在A3条件下的运作）
- 使"没有剩余的人"（劳动收入者）承担全部再生产成本，或选择不生育

### 2.3 ND11：认知污染——原债被A2语言吸收

市场化的配套叙事：
- "个人选择"：把结构性成本转嫁包装为自由选择
- "生育意愿下降"：把制度配置的必然输出解释为文化或心理问题
- "托育补贴"：用A2参数调整假装捕获了不可量化的再生产义务
- "生育奖励"：把N3原债货币化（这是ND11的典型操作）

**ND11判定**：这些叙事不改变原债结构，不恢复N3义务，只制造"问题已被政策处理"的虚假认知。

### 2.4 A3→A6闭环：有剩余的人买后代

市场化条件下，A6（血统ROI）在A3（货币完备性）语境下运作：

```
再生产成本全部私有化
    ↓
高资本收入者：再生产成本可覆盖，后代作为资产（A6投资）
低劳动收入者：再生产成本超过收入，生育成为纯损失
    ↓
生育成为资本配置行为，不是社会义务
    ↓
TFR向资本持有者集中，总体TFR崩溃
```

韩国TFR 0.72的结构：再生产成本私有化到极致 + 薪酬分化允许极端不平等 → 有剩余的少数人生育，没剩余的多数人不生。

## 三、主流方案的NAST诊断

| 方案 | NAST诊断 | 定性 |
|------|---------|------|
| 托育补贴 | A2参数调整，不改变成本私有化结构，产生ND11认知污染 | 症状管理 |
| 生育奖励/现金补贴 | 把N3原债货币化（A3侵入），ND11：假装原债可以被钱清偿 | N3违反 |
| 延迟退休 | 把再生产成本（养老）再次转嫁给劳动者，成本外部化继续 | A4变体 |
| 引进移民 | 在不改变结构的条件下替换人口，不恢复N3原债关系 | 症状管理 |
| 公共托育社会化 | 最接近N3义务恢复：把原债从个体归还给集体承担 | N3方向 |

**公共托育的NAST判定**：若满足N5防俘获条件（资源通道分离、受益者否决权、Sunset Clause），是N3义务的部分恢复。但N5循环标记依然有效：需要已有嵌入性承担萌芽才能防止国家收编。

## 四、规范性结论（骨架）

再生产成本市场化不是"个人选择的集合"，是N3义务的系统性转嫁。

TFR崩溃的结构性解释：
```
N3义务被私有化（市场化）
    ↓
再生产成本由个体全额承担
    ↓
低收入群体（r<g的一侧）退出生育
    ↓
TFR向资本持有者集中
    ↓
总体TFR崩溃 → CCST σ_b激活 → ℋ_b穿越风险
```
