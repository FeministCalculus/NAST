## 一、框架定位

### 1.1 银行/资本管理者的NAST定性

在司护制框架下，银行/资本管理者的角色是**steward（司护者）**，不是owner（占有者）。

判定依据（T4）：资本的真实受益者是使用资本的生产者和社群。管理者核心义务是维持资本在社群中的流通，不是从流通中攫取超额收益。

### 1.2 恢复能力连续谱（v5）

| 恢复类型 | 描述 | 案例 | 判定框架 |
|---------|------|------|---------|
| 自然恢复 | 内生物理机制 | 森林/河流/生态 | θ阈值（D2有物理意义） |
| 社会恢复 | 信用/劳动/制度机制 | 货币资源池 | 再循环能力 |
| 制度恢复 | 法律/政策干预 | 银行系统 | 外部化程度 |
| 无恢复 | 纯零和池子 | 理论极限 | 不适用 |

货币资源池属于"社会恢复"。

### 1.3 N3的适用边界

N3不适用于利息分析。N3处理对时间性存在的原债不可货币化清偿，不处理货币系统参数。

## 二、利息五成分解

| 成分 | 内容 | 再循环判定 |
|------|------|-----------|
| 时间偏好 | 现在消费vs未来消费的补偿 | 纳入总效应，对边缘借款人可能造成压力 |
| 风险补偿 | 覆盖违约损失 | 池子必要功能，通常不破坏 |
| 通胀补偿 | 覆盖货币贬值 | 货币系统特征，通常不破坏 |
| 运营成本 | 银行运作费用 | steward合理成本，通常不破坏 |
| 资本回报 | 占有/处置权带来的收益 | **核心判定**：r>g+归司护者时破坏再循环 |

**统一判定逻辑**：所有利息的总效应对边缘借款人是否造成再循环破坏，不逐个成分判定。

## 三、诊断层：N2判定（v7）

### 3.1 r>g推导链（工合形式化，SORRY-N2-1闭合）

设K=资本存量，Y=总产出，r=资本回报率，g=经济增长率，M=司护者，B=受益人，R_K=rK资本总回报，S=R_K/Y资本占产出份额。

**N2的形式化职位义务**：
- 合法收益：C_M = 维持资本流通的必要成本
- 非法收益：Π_M = R_K - C_M（超出必要成本的处置权收益）
- N2禁止Π_M > 0

**推导链**（r>g且资本回报归M时）：

```
S(t) = S(0) · e^{(r-g)t}
r > g → S(t)单调递增 → W(t)/Y(t)单调递减
→ 受益人再生产能力下降（劳动力再生产/生产性投资/偿债能力）
→ 再循环能力RC崩溃
→ N2职位违约
```

**关键限定**：r>g本身不是疾病。若资本回报归受益人（B），W包含资本收入，W/Y不必然下降，r>g不违反N2。

**正确条件**：~~r>g破坏再循环能力~~ ❌ → **r>g且资本回报归司护者时，破坏再循环能力** ✅

### 3.2 三子缺口闭合（v7）

**SORRY-N2-1a**（已闭合）：r>g是W/Y下降的充分条件（无对冲力量时），不是必然条件。有对冲力量（劳动组织、政策干预）时可减缓，但不消除趋势。

**SORRY-N2-1b**（已闭合）："资本回报归司护者"由法律/制度结构决定——steward获得超出管理费的资本回报（Π_M > 0）即构成资本回报归司护者。

| 制度结构 | 资本回报归属 | N2判定 |
|---------|------------|--------|
| 信托 | steward拿管理费，超额归受益人 | 不违反N2 |
| 公司 | 股东拿分红，管理层拿工资 | 取决于股东是否是steward |
| 银行 | 银行拿利润，存款人拿利息 | 潜在越权（视外部化程度） |
| 捐赠基金 | 基金会用回报执行使命 | 视外部化程度判定 |

**SORRY-N2-1c**（已闭合）：外部化程度由损失承担主体决定。

| 损失承担主体 | 外部化程度 | N2判定 |
|------------|-----------|--------|
| 基金/银行自身 | 完全内部化 | 不违反N2 |
| 借款人 | 部分外部化 | 可能违反N2 |
| 纳税人（公共救助） | 高度外部化 | 违反N2 |
| 社会（社会化损失） | 完全外部化 | 违反N2 |

### 3.3 规模敏感判定（v4-v7）

- **单案**：r>g但经济体可承受，且外部化已内部化 → 形式上合法
- **集体**：大规模复制r>g+资本归司护者模式 → 集体提取破坏再循环
- **阈值**：r>g是判定机制，外部化程度是权重

## 四、A3/A4接口（v5推导，v7不变）

N2越权的三特征与A4高度同构：
- N2处置权滥用 → A4结构性提取
- N2受益人损失 → A4从借款人提取超额价值
- N2成本外部化 → A4外部化违约风险

因此N2越权在Fc框架下的对应形式是A4。A3适用条件：利息被用于将原债/时间性存在货币化时构成A3污染。

## 五、案例分析（v7）

### 诺贝尔奖基金

**N2判定**：
- 若R_K被用于受益人（科学传承），且不外部化 → R_K不是司护者私有收益 → 形式上合法
- 若r>g且损失外部化给劳动者/自然 → 外部化存在 → 可能违反N2
- 集体：大规模捐赠基金复制此模式 → 集体效应破坏再循环

**循环论证修正**：不能用"寄生"预设结论。正确路径：r>g且外部化未内部化 → 违反N2；r>g且外部化已内部化 → 可能合法。

### 伊斯兰银行（Mudarabah）

银行承担亏损风险，利润共享。风险由steward承担而非转嫁借款人——可能不违反N2。争议：很多产品经济学上等价于利息，需逐产品判定是否破坏再循环。

### 私人信托

受托人拿管理费，超额归受益人。资本回报归受益人（B），不归steward（M），Π_M = 0。**最接近职位履行的金融形态，不违反N2。**

### 案例总结

| 案例 | 资本回报归属 | N2判定 |
|------|------------|--------|
| 诺贝尔奖 | 基金会用于使命 | 单案可能合法，集体效应待判定 |
| 伊斯兰银行（Mudarabah） | 银行承担风险，利润共享 | 可能不违反N2 |
| 私人信托 | 受益人 | 不违反N2 |
