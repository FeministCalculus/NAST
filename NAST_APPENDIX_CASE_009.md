## 一、框架定位

### 1.1 银行/资本管理者的NAST定性

在司护制框架下，银行/资本管理者的角色是**steward（司护者）**，不是owner（占有者）。

判定依据（T4受益者识别定理）：资本的真实受益者是使用资本的生产者和社群，不是资本管理者。管理者的职位由受益关系自动赋予，核心义务是维持资本在社群中的流通，不是从流通中攫取超额收益。

### 1.2 恢复能力连续谱（v5）

v3的二分法（自恢复系统 vs 资源池）已被连续谱替代：

| 恢复类型 | 描述 | 案例 | 判定框架 |
|---------|------|------|---------|
| 自然恢复 | 内生物理机制 | 森林/河流/生态 | θ阈值（D2有物理意义） |
| 社会恢复 | 信用/劳动/制度机制 | 货币资源池 | 再循环能力 |
| 制度恢复 | 法律/政策干预 | 银行系统 | 外部化程度 |
| 无恢复 | 纯零和池子 | 理论极限 | 不适用 |

**货币资源池属于"社会恢复"**：货币本身不恢复，但背后的经济系统（生产、劳动、信用）有某种恢复能力。

### 1.3 N3的适用边界

N3（原债不可货币化清偿）**不适用**于利息分析——N3处理对时间性存在的维护义务，不处理货币系统参数。这是N3适用边界的操作化示范。

## 二、利息五成分解

| 成分 | 内容 | 再循环判定（v5统一框架） |
|------|------|----------------------|
| 时间偏好 | 现在消费vs未来消费的补偿 | 对边缘借款人可能造成压力，纳入总效应判定 |
| 风险补偿 | 覆盖违约损失 | 池子必要功能，通常不破坏 |
| 通胀补偿 | 覆盖货币贬值 | 货币系统特征，通常不破坏 |
| 运营成本 | 银行运作费用 | steward合理成本，通常不破坏 |
| 资本回报 | 占有/处置权带来的收益 | **核心判定**：r>g时可能破坏再循环能力 |

**统一判定逻辑（v5）**：不逐个成分判定，而是**所有利息的总效应对边缘借款人是否造成再循环破坏**。即使单个成分"合理"，总效应破坏再循环仍违反N2。

## 三、诊断层：N2判定（v5）

### 3.1 r>g推导链（v6，从N2内部推出）

r>g不是外部输入。在零售金融市场的结构性假设下，从N2职位义务可以推导出r>g等同于N2职位违约：

| 步骤 | 命题 | 来源 | 状态 |
|------|------|------|------|
| 1 | N2职位义务：维持再循环能力 | NAST N2 | 成立 |
| 2 | 再循环能力要求：借款人偿还能力可持续 | 逻辑推导 | 成立 |
| 3 | 借款人偿还能力来自劳动收入（零售金融） | 结构性事实 | 成立（零售金融市场） |
| 4 | r>g → 劳动份额收缩 → 偿还能力下降 | 皮凯蒂实证（B权重） | 成立 |
| 5 | 偿还能力下降 → 违约率上升 → 再循环受损 | 逻辑推导 | 成立 |
| 6 | N2职位违约 | N2 + 上述推导 | 成立 |

**步骤3的assumption gap**：在零售金融/消费信贷/小微贷款市场中，借款人主要依赖劳动收入是可验证的结构性事实，不是断言。

**残余缺口**：机构间金融、高净值贷款市场的借款人可能有资本收入，r>g在这些市场不必然推出N2违约——因为偿还能力不依赖劳动份额。

- **单案**：r>g但经济体可承受（再循环未破坏）→ 形式上合法
- **集体**：大规模复制r>g模式 → 集体提取破坏再循环能力

## 四、A3/A4接口（v5，从断言改为推导）

### 从N2越权到A4的推导

N2越权的三特征：
1. 处置权滥用：steward用管理职位谋取私利
2. 受益人损失：steward的超额收益来自受益人的损失
3. 成本外部化：steward将风险/损失转嫁给借款人或社会

A4的特征：结构性提取 + 外部化违约风险

**映射**：N2越权三特征与A4高度同构，N2越权在Fc框架下的对应形式是A4。这不是两条路判定同一件事，是同一个判定在不同框架下的映射。

### A3的适用条件

利息使时间/货币关系可交易，是A3（货币完备性）的潜在通道。A3适用条件：利息被用于将原债/时间性存在货币化时构成A3污染；利息本身不一定是A3。

## 五、池化模型（v5）

```
资源池 → 成员借用 → 部分违约 → 池子覆盖损失 → 成员归还 → 池子维持（社会恢复）
              ↑
    费用判定（总效应框架）：
    维护基金/运营成本/风险储备 → 服务再循环能力维持 → 通常不破坏
    资本回报 → r>g时可能破坏再循环能力？→ 待论证（SORRY-N2-1）
```

## 六、案例分析

### 诺贝尔奖基金

**结构**：本金不动，只花投资回报，永续运作。依赖多元化投资组合（股票、债券、房地产）。

**NAST定性**：寄生于自恢复系统（经济体）的永续提取结构。不是自恢复系统，没有内生恢复机制。

**r>g判定**：
- 单案：r>g但经济体可承受 → 形式上合法
- 集体：大规模复制此模式，集体r>g可能破坏经济体再循环能力
- 这是规模敏感判定，不是单案判定

**循环论证风险修正（v5）**：不能直接用"寄生"预设结论。正确路径：若r>g且外部化未被内部化（损失转嫁给社会），则违反N2；若r>g但外部化已被内部化（损失由基金自身承担），则可能合法。

### 伊斯兰银行

**核心产品**：Mudarabah（利润-亏损分担）、Musharakah（合资）、Ijara（租赁）、Murabaha（成本加成）

**争议**：很多产品在经济学上等价于利息，只是重新包装。Mudarabah（银行承担亏损风险）若与传统利息有本质区别，可能是真正的池化模型；若只是标签游戏则不适用。

**NAST判定**：需区分"利润"和"利息"是否真的不同——判定标准是是否破坏再循环能力，不是名称。

### 私人信托

**结构**：受托人管理资产，回报归受益人，受托人收管理费，超额回报归受益人。

**NAST定性**：最接近职位履行的金融形态——steward（受托人）拿合理成本（管理费），超额不流向steward。

```
资产池 → 受托人管理 → 投资回报 → 管理费（合理成本）→ 超额归受益人
                          ↑
                   steward不拿超额 → 不构成越权
```

**[SORRY-利息-3]部分闭合**：私人信托提供了池化模型的参考结构。待补充：伊斯兰银行的实际运作是否与传统银行有本质区别。

## 七、规范性结论（v5）

```
v1：利息在公有制下不合法（错误）
v2：超额资本回报不合法（A2语言）
v3：超额资本回报是否破坏再循环能力？待判定
v4：单案可能合法，集体复制可能破坏再循环
v5：所有利息的总效应对边缘借款人是否造成再循环破坏？阈值：r>g

NAST不主张"消灭利息"，主张识别并剥离产生越权的成分。
```
