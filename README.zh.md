# NAST — 债管论 / Non-Appropriation Stewardship Theory

**版本**：v1.8-DRAFT  
**状态**：核心公理层稳定，未决SORRY 4项（3/7/8/9）

## 这是什么

NAST是一个诊断框架，分析市场逻辑（A2效用化约）如何系统性地无法计算破坏时间性存在的代价——生态系统、语言、社群生态、文化实践。

**核心主张**：管理义务来自受益关系，不来自产权。任何从时间性存在中受益并占据其延续环节的人，自动承担司护职位——不可转让、不可继承、不可放弃。这不是道德论证，是从受益关系本身推导出的结构性结论。

NAST没有目标状态（"自然应该是什么样的"）。它识别一条边界线：D2不可逆阈值——跨过去就回不来了。司护职位的内容是：保持系统在这条线以上，并向下一代传递。

## NAST诊断什么

**A2盲区**：当存在物满足以下任一条件时，A2无法陈述它——

- α（不可贴现）：存在意义分布在无限时间轴
- β（不可操作）：维护行为无法被货币化替代
- γ（不可识别）：A2无法建立输入-输出映射

不是"太有价值所以不能毁"，是**在A2语法里根本说不出来**。

**认知污染（ND11）**：A2用可量化替代指标假装捕获了不可量化损失，替代指标的存在本身关闭了追问。碳市场、ESG评分、空置税——问题不是定价不准确，是定价这个动作本身堵死了"该不该定价"的追问。

**A2→A3正反馈回路**：A2对不可量化之物完成定价 → A3将替代指标交易化 → 市场价格反过来为A2定价提供合法性 → 底层损失继续沉默。

---

## NAST不做什么

不提供目标状态。只画边界线，不说不准跨越。

不解决可规模化问题。嵌入性承担（真正的司护结构）需要小规模、具身、不可匿名的社群。匿名大社会是A2的载体，不是嵌入性承担的载体。这是确认的框架边界（SORRY-NAST-2），不是设计缺陷。

**诊断-处方离散边界定理**：对A2进行精确诊断的框架，必须使用A2无法翻译的语言。能规模化的处方必须进入A2可翻译的语言，进去就有被吸收的风险。没有中间态。
